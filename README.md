## Get started

1. **Clone the Repository**
   ```bash
   git clone https://github.com/Tediisu/Re-Spotify.git
   ```
2. **Navigate to the Project Directory**
   ```bash
   cd ./Re-Spotify
   ```
3. **Install dependencies**

   ```bash
   npm install
   ```

4. **Start the app**

   ```bash
   npx expo start
   ```
